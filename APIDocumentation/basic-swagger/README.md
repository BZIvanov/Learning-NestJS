# Basic Swagger Docs

This is a small NestJS demo project showing how to set up Swagger API documentation.

## Swagger UI

After running the app, open: `http://localhost:3000/api`.

This displays the interactive Swagger documentation for all endpoints.

## Official NestJS Swagger Docs

For more details, check the official documentation: [here](https://docs.nestjs.com/openapi/types-and-parameters).
