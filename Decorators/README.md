# Decorators

Decorators in Nest.js are metadata annotations used to tell Nest how classes, methods, or parameters should behave at runtime. They are a core part of Nest's architecture and help define things like routes, dependency injection, middleware behavior, guards, pipes, and more.

## Types of decorators in Nest.js

1. Class decorators
2. Method decorators
3. Parameter decorators
4. Property decorators
5. Custom decorators
